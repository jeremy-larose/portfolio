    maxiRose Blog

    The blog project for maxiRose was designed for speed and portability. Having a blog increases SEO and lends credibility to the company by providing a more personal touch to the site as well as another way to engage with users.

    This project utilizes ASP.NET and makes use of the functionality of Razor pages to quickly build out the front-end experience when combined with Bootstrap. By harnessing the power of ASP.NET, I was able to create and debug the blog with ease, which allowed me to complete the feature set requested by maxiRose. The project was linked to a Postgres server for ease of use with Entity Framework and Dependency Injection, and is hooked up for Continuous Integration/Deployment through linking Heroku and Github.

    The Garage

    The Garage was made for an automotive company that wanted a working demo of an application built using an ASP.NET back end with an Angular front end.

    This project connects to an EF Core InMemory database built with an API for access from anywhere. The client wanted the ability to monitor engine running status for their fleet of vehicles as well as handle upgrades and information regarding horsepower and torque statistics. The site was designed from the planning phase for maximum expandability, built from the ground up with the ability to add more information regarding the vehicles quickly. This project is currently set up for CI/CD to Microsoft Azure.
